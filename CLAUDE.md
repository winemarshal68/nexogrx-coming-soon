# nexogrx-coming-soon — orientation

This is a Cloudflare Pages static site bootstrapped by `nexo-create`.

- **Hosting:** Cloudflare Pages, project `nexogrx-coming-soon`
- **Repo:** `winemarshal68/nexogrx-coming-soon` on GitHub
- **Deploys:** push to `main` → `.github/workflows/deploy.yml` runs `wrangler pages deploy`
- **Secrets:** `CLOUDFLARE_API_TOKEN`, `CLOUDFLARE_ACCOUNT_ID` (set on the repo by the bootstrap)
- **Pages function:** `functions/_middleware.js` runs before every request (currently no-op)
- **Headers/redirects:** `_headers` and `_redirects` use Cloudflare Pages syntax
- **Sitemap:** auto-generated by `scripts/generate-sitemap.mjs` during the GitHub Action build

For Cloudflare API calls, use `$CLOUDFLARE_API_TOKEN` from the shell env (loaded from macOS Keychain by `~/.zshenv`). Don't `op read` for routine ops.
